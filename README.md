# c-interactive-menu

A utility menu built in C that offers several tools, including basic math operations, age classification, and weather-based recommendations.

## Features

- Interactive terminal menu  
- Input validation  
- Modular C functions  
- Math utility  
- Age classifier  
- Weather advice  
- Clean code structure  

## How to Run

Follow these steps to compile and execute the program:

1. Compile: `gcc main.c -o menu`
2. Run: `./menu`
